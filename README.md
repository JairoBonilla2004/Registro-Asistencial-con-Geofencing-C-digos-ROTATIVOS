# Registro-Asistencial-con-Geofencing-C-digos-ROTATIVOS
Aplicación móvil (Android) para registrar asistencia académica mediante QR y geofencing, con soporte web para que el docente inicie sesiones y proyecte los QR, usando Android Studio, Springboot para la api propia, Firebase, y autenticación mediante Google, Facebook y email/password.
